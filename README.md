# 🛡️ SafeMap — Anonymous Women Safety Platform

> Real-time incident reporting & AI-powered danger heatmap for women's safety

**Built for: Elite Her Community Hackathon 2026 | Team: Cypher**



## 🔴 The Problem

- 71% of women **never report** harassment due to fear & stigma
- Existing apps require login — killing trust and adoption
- No real-time map shows women **WHERE** it's unsafe
- Authorities can't act on what they don't know

## 💡 Our Solution

A web platform where women can **report, see, and avoid danger — anonymously.**
 Feature & Description 
 🔴 Anonymous Reporting : No login. No name. No fear. Report in seconds. 
 🗺️ Live Safety Heatmap : Real-time color-coded danger zones 
 🤖 AI Risk Prediction : ML detects patterns & warns about unsafe zones 
 🛣️ Safe Route Suggester : Navigate around high-risk areas 
 🔔 Live Alerts : Notify users near freshly reported incidents 

## 🚀 How It Works

1. Open SafeMap — no login required
2. Report incident — select location, type & time (fully anonymous)
3. Data saved — zero identity stored
4. Heatmap updates live for all users
5. AI analyzes patterns across reports
6. Alerts sent when zone risk score changes

## 🛠️ Tech Stack
 Frontend : HTML5, CSS3, JavaScript 
 Maps : Leaflet.js + CartoDB tiles 
 Heatmap : Leaflet.heat plugin 
 Database : Firebase Realtime DB  
 AI/ML : Python + Scikit-learn
